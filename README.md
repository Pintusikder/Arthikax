- Set up a profile name (e.g., "My Shop", "Personal")
- Choose a color theme (optional)
- Set a PIN for security (optional)

### 2. Add Transactions (Entry)
Navigate to **Entry** tab:
- Select transaction type: **Give** (Credit/à¤‰à¤§à¤¾à¤°) or **Take** (Debit/à¤œà¤®à¤¾)

Navigate to **History** tab:
- See all transactions with dates
- Filter by party name or date range
- Edit or delete entries
- View party-wise balances

### 4. Daily Cash Collection
Navigate to **Daily Hub** tab:
- Select your country and currency
- Enter count for each denomination
- Total calculates automatically
- Save collection for the day

- View overall summary (total give, take, balance)
- See party-wise balance sheet
- Export as PDF for printing
- Share invoices with parties

### 6. Settings & Customization

- Switch profiles or create new ones
- Change color theme
- Set/change PIN
- Export data
- Clear all data (with confirmation)
